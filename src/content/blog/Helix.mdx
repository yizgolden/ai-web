## 简介

使用任何数据创建您自己的人工智能，根据您的需求构建和优化文本和图像人工智能。根据您的数据进行训练、微调和生成。

## 特点

1. 开放式配重：保持对数据和从中派生的模型的完全控制。由于我们使用开源模型，因此您可以拥有权重、保持控制并简化数据治理。消除供应商锁定，提高延迟和安全性。
2. 人性化：通过自然的聊天界面进行数据准备和微调工作流程，您可以微调高性能、优化的小型模型，这些模型在特定任务中表现出色。这使得每个人都可以训练自己的 AI。
3. 开发人员和DevOps友好：Helix 易于集成：我们有一个简单的 API，您还可以查看从聊天应用程序内部进行哪些 API 调用。使用我们优化的容器可以轻松部署堆栈，使用 NVIDIA Docker 或 RunPod 部署 GPU 运行器。
