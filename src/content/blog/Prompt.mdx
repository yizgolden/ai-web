## 怎么样写好一个 Prompt？

我们在写 Prompt 时，需要考虑以下几个方面：

1. 清晰明确的问题：确保您的问题具有明确的意图和清晰的描述，这样机器人才能更好地理解您的需求。
2. 上下文提供：如果您需要机器人对之前的对话进行回应，可以提供相关的上下文信息，以帮助机器人更好地理解并作出恰当的回答。
3. 限制回答范围：如果您对某个特定主题或领域有特定的问题，可以在 Prompt 中明确指定，以便机器人针对该主题提供准确的回答。
4. 明确禁止的主题：如果您希望机器人避免回答某些敏感话题或涉及违法内容的问题，请在 Prompt 中明确说明，以确保机器人不会提供相关信息。
5. 表达方式：在编写 Prompt 时，尽量使用清晰简洁的语言，避免歧义或模糊性，以确保机器人可以准确理解您的问题。
