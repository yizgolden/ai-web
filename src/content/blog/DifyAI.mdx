## 简介

新一代大型语言模型应用开发平台，轻松构建和运营生成式 AI 原生应用。基于任意大型语言模型构建企业级 Assistants API 和 GPTs，原腾讯云 CODING DevOps 团队的核心成员倾力打造。

Dify 是一个易于使用的 LLMOps 平台，提供了多种应用程序类型和模板，包括 AI 聊天机器人、代码转换器、SQL 生成器、新闻内容编写、创意脚本等。团队使用 Dify，可以基于 GPT-4 等模型快速「开发+部署」 AI 应用程序并进行可视化操作，还可以将程序对外进行发布。Dify 这个名字来源于 Define 和 Modify 这两个词，Dify 可以理解为 Do it for you，代表了帮助开发人员不断改进其 AI 应用程序的愿景。

## 特点

1. 可视化编排 Prompt，通过界面化编写 prompt 并调试，只需几分钟即可发布一个 AI 应用。
2. 接入长上下文（数据集），全自动完成文本预处理，使用您的数据作为上下文，无需理解晦涩的概念和技术处理。
3. 基于 API 开发，后端即服务。您可以直接访问网页应用，也可以接入 API 集成到您的应用中，无需关注复杂的后端架构和部署过程。
4. 数据标注与改进，可视化查阅 AI 日志并对数据进行改进标注，观测 AI 的推理过程，不断提高其性能。
5. 免费版：200次GPT免费试用，团队成员1，构建应用程序10，矢量存储5MB，留言请求每天 500 个注释配额限制10，日志历史记录15天
